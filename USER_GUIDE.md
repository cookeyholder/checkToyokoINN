# 東橫 INN 空房監控系統 - 使用者指南

## 1. 系統概述

- 自動監控東橫 INN 房間可用性
- 符合時間條件時發送郵件通知
- 支援多分店、多房型監控

## 2. 基本操作

### 2.1 建立提醒

1. 選擇分店
2. 選擇房型 (可複選)
3. 設定入住/退房日期
4. 設定監控時間範圍 (HH:MM 格式)
5. 點擊「建立提醒」

### 2.2 管理提醒

- **暫停**: 停止監控,不發送通知
- **啟用**: 恢復監控
- **歷史**: 檢視檢查記錄和通知歷史
- **刪除**: 永久刪除提醒

### 2.3 批量操作

- 支援一次建立多個房型的提醒
- 每個房型建立一筆獨立提醒
- 支援同時監控不同房型和時間

## 3. 進階功能

### 3.1 時間篩選

- **提醒開始時間**: 監控何時開始檢查
- **提醒結束時間**: 監控何時停止檢查
- 支援跨日時間範圍 (如 23:00-01:00)

### 3.2 通知機制

- 有空房時發送郵件通知
- 同一提醒最多 1 小時發送一次通知
- 過期提醒自動停止檢查

### 3.3 檢查歷史

- 檢視每次房間檢查的結果
- 確認是否成功發送通知
- 查看錯誤訊息 (若有)

## 4. 常見問題

### Q: 為什麼沒有收到通知？

A: 
1. 檢查郵件地址是否正確
2. 檢查提醒狀態是否為「啟用」
3. 檢查當前時間是否在監控時間範圍內
4. 查看檢查歷史確認是否檢查到有空房

### Q: 可以同時監控多少個提醒？

A: 沒有限制,但每個提醒每 5 分鐘檢查一次

### Q: 過期提醒會怎樣？

A: 系統會自動停止檢查,但不會刪除記錄
